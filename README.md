<h2>🛠️ Installation Steps:</h2>

<p>1. docker pull</p>

```
docker pull thescriptsage/shortifynetapi
```

<p>2. docker run</p>

```
docker run -d -p 5000:80 thescriptsage/shortifynetapi
```
